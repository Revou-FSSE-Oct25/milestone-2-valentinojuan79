# RevoFun Studio | Gaming Company

## Overview
RevoFun Studio is a gaming company that provides a simple web games to give you a short break from your busy life.
It's cheaper than therapy and way more fun than doom-scrolling.


## Features Implemented
### 1. Home Page
Introduces gaming company and showcases featured game.
### 2. Library
Displays all the games RevoFun has created, as well as upcoming titles currently in development.
### 3. Games
Currently, RevoFun offers 3 fully playable games. Number Guessing, Rock Paper Scissors, and Speed Clicker.
### 4. JavaScript
Provides the underlying logic and mechanics for each game.
### 5. CSS
Handles the styling and ensures the layout is responsive across all devices.


## Technologies Used
* HTML5
* CSS
* Tailwind CDN
* JavaScript
* Google Fonts : Inter, Bungee
* Gemini AI : Generate Images
